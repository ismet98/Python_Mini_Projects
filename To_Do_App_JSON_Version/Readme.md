
# H1 To Do App JSON File Documentation

The following folder presents an updated version of the To Do App, incorpoorating uploading, and saving tasks lists on to a JSON file. 

The versions and the implemented changes are as follows: 

# H2 1. To Do App JSON V1

This was a version that incorperated the basic elements of JSON onto the application. It did this by seeing the task list as a single dictionart, where a task itsef could be added into it. As a result, printing the list would only by printing the different items across the dictionary, which were numbered in order

# H2 2. To Do App JSON V2

This design revolved around seeing the data structure as a list of dictionaries, simultaenously tracking the status of the task list. The user would be able to see the task list and then choose to keep its status as done or not done. This additional layer of complexity provides more freedom for users to interact with their tasks list
